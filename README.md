#  Sistema Gestor de Expedientes Clínicos

Proyecto universitario desarrollado como sistema web para la **gestión y administración de expedientes clínicos** de alumnos de una institución educativa, permitiendo el registro, consulta, edición y eliminación de pacientes, así como la administración de usuarios con distintos roles.

---

##  Descripción del Proyecto

El sistema permite a **administradores y médicos** gestionar expedientes clínicos de manera digital, segura y organizada.  
Incluye control de acceso por roles, manejo de base de datos y carga masiva de información mediante archivos Excel.

---

##  Roles de Usuario

- **Administrador**
  - Crear, editar y eliminar usuarios
  - Visualizar todos los pacientes
  - Gestionar expedientes clínicos

- **Médico**
  - Registrar pacientes
  - Consultar y editar expedientes clínicos
  - Importar pacientes desde Excel

---

##  Funcionalidades Principales

- Registro e inicio de sesión de usuarios
- Control de acceso por roles (admin / médico)
- Registro de pacientes con datos clínicos
- Visualización de pacientes en tablas dinámicas
- Búsqueda de pacientes en tiempo real
- Edición y eliminación de pacientes
- Importación de pacientes desde archivos Excel
- Interfaz web con HTML y CSS

---

##  Tecnologías Utilizadas

- **Node.js**
- **Express.js**
- **MySQL**
- **HTML5**
- **CSS3**
- **JavaScript**
- **Multer** (subida de archivos)
- **xlsx** (lectura de archivos Excel)
